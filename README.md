# Molecule Project

## Tagline
Organize Your Thoughts Visually

## Features
- Sticky Notes
- Drawing Tools
- Highlighting
- Magnifier
- Local Storage
- Lightning Fast Performance
- Private Offline-First
- Dark/Light Theme

## Tech Stack
- Frontend: React, Redux
- Backend: Node.js, Express
- Database: MongoDB
- Others: HTML5, CSS3, WebSockets
